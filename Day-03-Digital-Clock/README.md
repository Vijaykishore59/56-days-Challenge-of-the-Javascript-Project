# ⏰ Day 4 - Digital Clock

A clean and responsive **Digital Clock** built using **HTML, CSS, and JavaScript** as part of my **56 Days JavaScript Projects Challenge**.

---


## 🚀 Features

- 🕒 Live digital clock
- 📅 Displays the current date
- ⚡ Updates automatically every second
- 🎨 Modern gradient background
- 💎 Clean and responsive UI
- 📱 Responsive design
- 🌐 Beginner-friendly project

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

---

## 📂 Project Structure

```text
Day-04-Digital-Clock/
│── index.html

│── script.js
│── preview.png
└── README.md
```

---

## 📖 What I Learned

- Working with the JavaScript `Date` object
- Creating reusable functions
- Updating the DOM using `innerText`
- Formatting time with leading zeros
- Using conditional statements (`if`)
- Using `setInterval()` to update the clock every second
- Styling layouts using Flexbox
- Building responsive user interfaces with CSS

## ▶️ How to Run

1. Clone the repository.

```bash
git clone https://github.com/Vijaykishore59/56-days-Challenge-of-the-Javascript-Project.git
```

2. Navigate to the project folder.


4. Double-click **index.html** or open it with Live Server in VS Code.
