# 🎯 Day 04 - To-Do List App

Welcome to **Day 04** of my **56 Days JavaScript Projects Challenge**.

In this project, I built a fully functional **To-Do List App** using **HTML, CSS, and JavaScript**. This project focuses on strengthening JavaScript fundamentals through DOM manipulation, event handling, dynamic element creation, and state management.

---

## 📖 Project Overview

The To-Do List App allows users to efficiently manage their daily tasks. Users can add new tasks, mark them as completed, delete tasks, and monitor their progress with real-time task counters.

This project helped me gain hands-on experience with creating dynamic user interfaces and manipulating the DOM using vanilla JavaScript.

---

## 🚀 Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- 🗑️ Delete tasks
- 📊 Live task statistics
  - Total Tasks
  - Completed Tasks
  - Pending Tasks
- ⚠️ Input validation to prevent empty tasks
- 🎨 Clean and responsive user interface

## ⚙️ How It Works

1. Enter a task in the input field.
2. Click the **Add** button.
3. A new task is dynamically added to the list.
4. Use the checkbox to mark tasks as completed.
5. Click the delete button to remove tasks.
6. Task counters update automatically whenever tasks are added, completed, or deleted.
