# 🧮 Day 01 - Calculator Application

Welcome to **Day 01** of my **56 Days Coding Challenge**.

In this project, I built a fully functional **Calculator Application** using **HTML, CSS, and JavaScript**. This project focuses on strengthening the fundamentals of JavaScript by implementing calculator logic, DOM manipulation, and event handling.

---

# 📖 Project Overview

The Calculator Application is designed to perform basic arithmetic operations with a clean and responsive user interface.

This project helped me practice:

- HTML Structure
- CSS Grid Layout
- JavaScript Fundamentals
- DOM Manipulation
- Event Handling
- Error Handling
- Calculator Logic

---

# ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧹 Clear Display
- ⌫ Backspace Functionality
- 📱 Responsive Design
- ⚡ Interactive Button Hover Effects
- 🚫 Error Handling for Invalid Expressions
