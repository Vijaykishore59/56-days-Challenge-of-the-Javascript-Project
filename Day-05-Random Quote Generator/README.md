# 🎯 Day 05 - Random Quote Generator

Welcome to **Day 05** of my **56 Days JavaScript Challenge**.

In this project, I built a **Random Quote Generator** using **HTML, CSS, and JavaScript**. The application fetches inspirational quotes from a public API and displays a new random quote each time the user clicks the button. This project helped me understand asynchronous JavaScript concepts such as **Fetch API**, **Promises**, **Async/Await**, and **DOM Manipulation**.

---

## 📖 Project Overview

The Random Quote Generator provides users with a new inspirational quote on demand. It communicates with an external API, retrieves quote data, and dynamically updates the webpage without reloading.

---

## ✨ Features

- 🎲 Generate a random quote
- 🌐 Fetch data from a public REST API
- 👤 Display the author's name
- ⚡ Asynchronous data fetching using Async/Await
- 🎨 Modern and responsive user interface
- 🔄 Generate unlimited random quotes
- 📱 Mobile-friendly layout

## 🚀 How It Works

1. The webpage loads.
2. A random quote is requested from the API.
3. The API returns quote data in JSON format.
4. The quote and author's name are displayed.
5. Clicking the **New Quote** button fetches another random quote instantly.
