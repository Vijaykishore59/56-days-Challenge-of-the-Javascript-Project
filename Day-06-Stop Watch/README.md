# ⏱ Day 06 - Stopwatch

A simple and responsive **Stopwatch Application** built using **HTML, CSS, and JavaScript** as part of my **56 Days JavaScript Challenge**.

---

## 🚀 Live Features

* ▶️ Start the stopwatch
* ⏸️ Pause the stopwatch
* 🔄 Reset the stopwatch
* 🏁 Record lap times
* 📋 Display lap history
* 📱 Responsive user interface

## 🎯 How It Works

1. Click **Start** to begin the stopwatch.
2. Click **Pause** to stop the timer temporarily.
3. Click **Reset** to reset the timer and clear all lap records.
4. Click **Lap** to save the current stopwatch time.
