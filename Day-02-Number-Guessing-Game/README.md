# 🎯 Number Guessing Game

A fun and interactive Number Guessing Game built using **HTML**, **CSS**, and **Vanilla JavaScript**. The player has **10 attempts** to guess a randomly generated number between **1 and 100**.

## 📌 Features

- 🎲 Random number generated between **1 and 100**
- 🔢 User input validation
- 📉 Tracks remaining attempts
- 📝 Displays previous guesses
- 🎉 Win message when the correct number is guessed
- 😢 Game Over message after all attempts are used

## 🎮 How to Play

1. Enter a number between **1 and 100**.
2. Click the **Check Guess** button.
3. The game will tell you whether your guess is:
   - 📈 Too High
   - 📉 Too Low
   - ✅ Correct
4. You have **10 attempts** to guess the correct number.
5. If you guess correctly, you win and can restart the game.
6. If all attempts are used, the game ends and reveals the secret number.
